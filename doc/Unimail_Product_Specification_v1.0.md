# Unimail Product Specification v1.0

## 1. 产品概述

### 产品名称

Unimail

Slogan:

> One Inbox. Every Email.

### 产品定位

Unimail 是一款
Local-first（本地优先）的跨邮箱桌面客户端，让用户在一个安全的软件中管理
Gmail、Outlook、QQ邮箱、163邮箱等多个邮箱。

核心理念：

-   邮件数据默认保存在用户本地
-   不依赖第三方云服务器
-   用户完全掌控自己的邮件数据

# 9. 邮箱接入方案

## Gmail

使用：

-   OAuth2
-   Gmail API

流程：

用户授权 -\> 获取Token -\> 本地保存 -\> 同步邮件

## Outlook

使用：

-   Microsoft Graph API

## QQ邮箱 / 163邮箱

使用：

-   IMAP
-   SMTP

用户开启服务并提供授权码。

------------------------------------------------------------------------

# 10. 安全设计

## Token安全

禁止明文保存。

使用：

Windows:

-   DPAPI

macOS:

-   Keychain

## 数据库安全

使用：

-   SQLCipher

对本地数据库加密。

------------------------------------------------------------------------

# 11. 邮件同步

## 首次同步

默认：

最近500封邮件。

流程：

    Fetch

    ↓

    Parse

    ↓

    Save SQLite

    ↓

    建立索引

------------------------------------------------------------------------

## 增量同步

使用：

IMAP UID。

例如：

    last_uid = 1000

    服务器新增:

    1001-1050

    只同步新增邮件
